A custom Fabric moderation utility for Mineberry workflows.

Open a full moderation panel instantly from chat and handle actions, alt checks, active punishments, and history from one place.

---

## Quick Use

```text
.m <username>
```

This opens the moderation panel for that player.

---

## Main Features

### Unified Moderation Panel
- Single-screen moderation workflow
- Fast pivot between related accounts
- Built for high-speed in-game moderation

### Moderation Actions
- Warn / Mute / Ban flows
- Reason presets + custom reasons
- Duration and server scope options
- Silent action toggle
- Command preview + one-click copy

### Alt Intelligence
- Auto-detects and displays alts by status
- Separate columns for:
  - Banned
  - Online
  - Offline
- Click any alt to open that user panel directly

### Active Punishment Checks
- Active Mutes section
- Active Bans section
- Expandable detail views
- Quick copy support

### History Browser
- View punishment history inside panel
- Styled readable entries
- Page navigation with back/next controls
- Refresh + copy support

### Multi-Command Action Queue
- Handles multi-target actions cleanly
- Sends chained commands with safe spacing

### Clean Chat Experience
- Lookup output suppression while panel checks run
- Keeps public chat readable during moderation lookups

### QoL Shortcuts
- `.` opens chat prefilled with `.`
- Standard `/` behavior remains unchanged
